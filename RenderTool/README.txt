Fast and mobile friendly WebGL renderer tool. Demo avaliable on:
http://kondzio9.ayz.pl/rend/


Implemented features:
- mouse click/touch rotation
- scroll/two finger zoom
- Autorotation X
- FPS counter
- model grabbing

*********** IMPORTANT ***********

To convert 3D models to .json use Blender and io_three:
https://github.com/repsac/io_three


At some browsers (eg. chrome) the code must be run on 
localserver/server due to security restrictions, 
(problem with getting and setting texture)
